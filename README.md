# LDA (Lego Design Aid)

이미지·그림·사진을 업로드하면
레고 브릭 언어로 다시 설계해주고,
필요한 브릭 구성과 조립 흐름까지 제안해주는
레고 아이디어 설계 보조 웹 서비스입니다.

- 개인 취미(레고)를 기반으로 한 AI + 웹 토이 프로젝트
- 레고 아이디어스 스타일의 커스텀 제품 시뮬레이터를 장기 목표로 합니다.
- 어린이/입문자부터, 자신만의 레고 아이디어를 표현하고 싶은 크리에이터까지를 대상으로 합니다.

## 폴더 구조 (초안)

- frontend/ : 프론트엔드 (React + Vite 예정)
- backend/  : 백엔드 (Python + FastAPI 예정)
